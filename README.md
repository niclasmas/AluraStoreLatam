# AluraStoreLatam
Entrega final Desafío Alura

Análisis de Ventas: Alura Store
Este proyecto analiza el rendimiento de 4 tiendas para decidir cuál debe vender el Sr. Juan basándose en datos reales.

Propósito
Identificar la tienda menos rentable evaluando ingresos, satisfacción del cliente y costos de envío.

Estructura
AluraStoreLatam.ipynb: Cuaderno con el código y análisis.

README.md: Documentación del proyecto.

Hallazgos Clave
Ingresos: La Tienda 4 tiene las ventas más bajas.

Clientes: La Tienda 4 tiene la peor calificación promedio.

Logística: La Tienda 4 tiene el costo de envío más caro.

Conclusión: Se recomienda vender la Tienda 4 por su baja rentabilidad y altos costos operativos.

Cómo usarlo
Descarga el archivo .ipynb.

Súbelo a Google Colab.

Carga los archivos .csv de las tiendas y ejecuta todas las celdas.
